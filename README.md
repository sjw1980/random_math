# 덧셈 문제 생성기

A4 출력용 덧셈 문제를 난이도별로 생성하는 스크립트입니다. 이미지(`.png`)가 `output/` 폴더에 저장됩니다.

## 난이도 규칙 (현재 구현)

**1, 3: 쉬움 (받아올림 없는 연습)**

- 숫자 개수: 2개
- 범위: 각 1~30
- 조건: 두 숫자의 일의 자리 합이 10 이하 → 받아올림 없음

**2: 100단위/20단위/한자리 혼합 규칙**

- 숫자 개수: 2개 또는 3개 (무작위)
- 100단위 숫자(100, 200, …, 900)는 최대 1개 (없어도 됨)
- 1의 자리 숫자(1~9)는 최대 1개 (없어도 됨)
- 나머지는 20단위 숫자(20, 40, 60, 80)로 채움

- **4: 큰 수 + 작은 수 조합**

  - 숫자 개수: 2개
  - 한 숫자: 100~149
  - 다른 숫자: 10단위(10, 20, …, 90) 또는 1단위(1~9)

- **5-6: 기본(처음 만든 규칙)**

  - 숫자 개수: 2개 또는 3개 (무작위)
  - 범위: 각 1~100

- **7-10: 어려움**
  - 숫자 개수: 3개
  - 범위: 각 50~100

모든 경우 큰 수가 왼쪽에 오도록 정렬됩니다.

## 실행 방법

- 기본 난이도 3: `python generate_problems.py`
- 특정 난이도: `python generate_problems.py 4` (1~10 입력 가능)

## 출력

- A4 크기, 300 DPI PNG
- 2열 × 15문제, 총 30문제
- 파일명 예: `output/addition_problems_lv4_YYYYMMDD_HHMMSS.png`

## 필요 패키지

- Pillow (`pip install pillow`)

## 참고

- `output/` 폴더는 `.gitignore`에 포함되어 있습니다.
