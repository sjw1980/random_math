# 덧셈 문제 생성기

A4 출력용 덧셈 문제를 난이도별로 생성하는 스크립트입니다. 이미지(`.png`)가 `output/` 폴더에 저장됩니다.

## 난이도 규칙 (현재 구현)

**1, 3: 쉬움 (받아올림 없는 연습)**

- 숫자 개수: 2개
- 범위: 각 1~30
- 조건: 두 숫자의 일의 자리 합이 10 이하 → 받아올림 없음

**2: 100단위/20단위/한자리 혼합 규칙**

- 숫자 개수: 2개 또는 3개 (무작위)
- 100단위 숫자(100, 200, …, 900)는 최대 1개 (없어도 됨)
- 1의 자리 숫자(1~9)는 최대 1개 (없어도 됨)
- 나머지는 20단위 숫자(20, 40, 60, 80)로 채움

- **4: 큰 수 + 작은 수 조합**

  - 숫자 개수: 2개
  - 한 숫자: 100~149
  - 다른 숫자: 10단위(10, 20, …, 90) 또는 1단위(1~9)

- **5-6: 기본(처음 만든 규칙)**

  - 숫자 개수: 2개 또는 3개 (무작위)
  - 범위: 각 1~100

- **7-10: 어려움**
  - 숫자 개수: 3개
  - 범위: 각 50~100

모든 경우 큰 수가 왼쪽에 오도록 정렬됩니다.

## 실행 방법

- 기본 난이도 3: `python generate_problems.py`
- 특정 난이도: `python generate_problems.py 4` (1~10 입력 가능)

## 출력

- A4 크기, 300 DPI PNG
- 2열 × 18문제, 총 36문제
- 파일명 예: `output/addition_problems_lv4_YYYYMMDD_HHMMSS.png`

## 필요 패키지

- Pillow (`pip install pillow`)

## 숫자 읽기 문제 생성기 (`generate_reading.py`)

- 목적: 주어진 자릿수에 맞는 난수를 생성하고 그 숫자를 한글로 표기한 문제(예: `213` → `이백십삼`)를 만든 뒤, 오른쪽에 답 입력란을 배치한 A4용 이미지를 생성합니다.
- 사용법: `python generate_reading.py <digits> [count]`
  - `<digits>`: 생성할 숫자의 자릿수(1~4). 예: `3` → 100~999 범위에서 생성.
  - `[count]`(선택): 생성할 이미지 개수(기본값 1).
- 출력:
  - 각 이미지에 36문제(2열 × 18행)를 배치합니다.
  - 파일명 예: `output/reading_problems_3digits_YYYYMMDD_HHMMSS.png`
  - 출력 형식: A4, 300 DPI PNG
- 폰트: 한글 표기를 위해 시스템의 한글 지원 글꼴(`malgun.ttf` 또는 `NanumGothic.ttf`)을 우선 사용합니다. 해당 폰트가 없으면 기본 폰트로 폴백되므로, Windows 환경에서는 `malgun.ttf`가 정상 설치되어 있는지 확인해주세요.

## 참고

- `output/` 폴더는 `.gitignore`에 포함되어 있습니다.
